---|---|---|---|---|---  
`ym:s:adfoxRequests` |  Запросы кода Adfox |  Запросы кода Adfox со страниц, по которым был ответ от сервера Adfox: загрузки баннеров и загрузки по умолчанию. Работает по множеству `Adfox`. |  `int` |  есть |  2019-03-20  
`ym:s:adfoxRenders` |  Отрисовки Adfox |  Запросы кода, по которым была засчитана загрузка баннера. Работает по множеству `Adfox`. |  `int` |  есть |  2019-03-20  
`ym:s:adfoxRendersDefault` |  Загрузки по умолчанию |  Количество запросов кода, по которым были произведены показы баннеров по умолчанию. Баннеры по умолчанию показываются в том случае, если нет для показа баннеров из коммерческих рекламных кампаний в силу настроек размещения, ограничений по количеству показов, скорости открутки, таргетирований. Работает по множеству `Adfox`. |  `int` |  есть |  2019-03-20  
`ym:s:adfoxRendersDefaultShare` |  Доля загрузок по умолчанию |  Доля запросов кода, по которым были произведены показы баннеров по умолчанию. Баннеры по умолчанию показываются в том случае, если нет для показа баннеров из коммерческих рекламных кампаний в силу настроек размещения, ограничений по количеству показов, скорости открутки, таргетирований. Работает по множеству `Adfox`. |  `percents` |  есть |  2019-03-20  
`ym:s:adfoxShows` |  Показы Adfox |  Зафиксированный системой показ рекламного баннера пользователю. Количество засчитанных показов может отличаться от количества загрузок баннеров, если в рекламной кампании переопределена методика учета показов. Работает по множеству `Adfox`. |  `int` |  есть |  2019-03-20  
`ym:s:adfoxClicks` |  Клики Adfox |  Количество кликов по рекламным баннерам. Работает по множеству `Adfox`. |  `int` |  есть |  2019-03-20  
`ym:s:adfoxVisibility` |  Процент видимости Adfox |  Доля засчитанных показов в общем количестве отрисовок. Отрисовки — запросы кода, по которым была засчитана загрузка баннера. Работает по множеству `Adfox`. |  `percents` |  есть |  2019-03-20  
`ym:s:adfoxRendersPerUser` |  Отрисовки на посетителя Adfox |  Среднее количество загрузок баннеров Adfox (в том числе РСЯ) на одного посетителя. Работает по множеству `Adfox`. |  `double` |  есть |  2019-03-20  
`ym:s:adfoxRendersPerVisit` |  Отрисовки на визит Adfox |  Среднее количество загрузок баннеров Adfox (в том числе РСЯ) на один визит. Работает по множеству `Adfox`. |  `double` |  есть |  2019-03-20  
`ym:s:adfoxRendersPerHit` |  Отрисовки на просмотр Adfox |  Среднее количество загрузок баннеров Adfox (в том числе РСЯ) на один просмотр страницы. Работает по множеству `Adfox`. |  `double` |  есть |  2019-03-20  
`ym:s:adfoxPrice` |  Суммарный доход Adfox |  Расчетный суммарный доход от показов рекламы. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxCPMV` |  CPMV Adfox |  Cost per mille viewable ads, расчетная стоимость одной тысячи показов. В рублях без НДС. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxECPM` |  eCPM Adfox |  Effective cost per mille, стоимость одной тысячи отрисовок рекламных блоков. В рублях без НДС. Отрисовки — запросы кода, по которым была засчитана загрузка баннера. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxRPM` |  RPM Adfox |  Revenue per mille, расчетная стоимость одной тысячи запросов кода Adfox — включая те запросы, по которым не удалось подобрать рекламу. В рублях без НДС. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxARPU` |  ARPU Adfox |  Средний расчетный доход от показов рекламы на одного посетителя. Работает по множеству `Adfox`. |  `currency` |  есть |  2018-03-20  
`ym:s:adfoxPricePerUser` |  Расчетный доход на посетителя Adfox |  Средний размер вознаграждения за показ блоков РСЯ в расчете на одиного пользователя. В рублях без НДС. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPricePerVisit` |  Расчетный доход визита Adfox |  Средний расчетный доход на один визит. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPricePerHit` |  Расчетный доход просмотра Adfox |  Средний размер вознаграждения в расчете на один просмотр страницы. В рублях без НДС. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPriceStrict` |  Расчетный доход, Прямые продажи |  Ориентировочная стоимость показов рекламы, размещенной на площадке напрямую. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPriceYan` |  Расчетный доход, РСЯ |  Суммарный доход за показы рекламных блоков РСЯ. В рублях без НДС. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPriceGoogle` |  Расчетный доход, Google |  Ориентировочная стоимость показов рекламных блоков Google. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20  
`ym:s:adfoxPriceHeaderBidding` |  Расчетный доход, Header bidding |  Ориентировочная стоимость показов рекламных баннеров Header Bidding. Работает по множеству `Adfox`. |  `currency` |  есть |  2019-03-20
